### Description

Replaces the coordinate transformation matrix by a coordinate transformation matrix that ``exactly undoes the original transformation''. For example, if the original transformation was ``scale by 2 and then shift right by 1cm'' the new one is ``shift left by 1cm and then scale by $1/2$''. This command will produce an error if the determinant of the matrix is too small, that is, if the matrix is near-singular. % %
