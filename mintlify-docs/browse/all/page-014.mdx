## 2903. Environment

**ID**: `8c31dfc9d77f`  
**Type**: environment  
**Package**: pgfplots  

### Description

options} The |smithchart| environment draws Smith charts. It accepts the same options as |axis|. In fact, it is equivalent to |axis[|options|,axis type=smithchart]|. codeexample[] tikzpicture smithchart[ title=Impedance Smith Chart, ] coordinates { (0.5,0.2) (1,0.8) (2,2) }; smithchart tikzpicture codeexample The example above visualizes three data points using the initial configuration of Smith charts; the data points are interpreted as complex numbers $z = x + j y$ and are mapped using $r(z)$.
