## 1603. Command Specification

**ID**: `67a5acbd5faa`  
**Type**: command_specification  
**Package**: tikz  

### Description

color modelinitial colorstep} This command creates a new style sheet using ||. This style sheet will only have a default style setup that maps numbers to the color in the color series starting with initial color and having a stepping of step. Note that when the value of the attribute is |1|, which it is the first data set, the second color in the color series is used (since counting starts at |0| for color series). Thus, in general, you need to start the initial color ``one early''. % %
