## 1465. Command

**ID**: `90623d180263`  
**Type**: command  
**Package**: pgfplots  

### Description

Provide || as trailing path commands after || to draw a closed line from the last plot coordinate to the first one. Use || whenever you intend to fill the area under a plot. codeexample[] tikzpicture axis {x^2+2} ; axis tikzpicture codeexample codeexample[] tikzpicture axis + [fill] {x^2+2} ; axis tikzpicture codeexample In case of stacked plots, || connects the current plot with the previous plot instead of connecting with the $x$-axis.The implementation for stacked plots requires some addition
