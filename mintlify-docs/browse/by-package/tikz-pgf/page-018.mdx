**Type**: command  
**Package**: tikz-pgf  

### Description

} This command will shortened the start of every stroked path by the given dimension. This shortening is done in addition to automatic shortening done by a start arrow, but it can be used even if no start arrow is given. It is usually better to use the |sep| key with arrow tips. This command is useful if you wish arrows or lines to ``stop shortly before'' a given point. codeexample[preamble={arrows.meta}] pgfpicture {5mm} stroke Latex- 4pt {5mm{0cm}} {2cm{0cm}} stroke pgfpicture codeexample
