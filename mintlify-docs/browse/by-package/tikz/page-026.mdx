### Description

offset} This command computes the following point: Consider the curve last installed using the command ||. We travel along this curve by distance, arriving at a point $p$. Then, we turn by $90^$ and travel by offset units ``always from the curve'', arriving at a point $q$. This point $q$ will now be returned in |@x| and |@y|; furthermore, the transformed local coordinate system at point $q$ will also be returned |@xa| and the other registers, see || for details. % %

\makeatletter
\def\polartransformation{
  % \pgf@x will contain the radius
  % \pgf@y will contain the distance
  \pgfmathsincos@{\pgf@sys@tonumber\pgf@x}%
  % pgfmathresultx is now the cosine of radius and

}
