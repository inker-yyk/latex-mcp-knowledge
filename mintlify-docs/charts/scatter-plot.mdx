## Overview

Scatter plots display data points in a two-dimensional space, useful for showing correlations and distributions.

## Basic Scatter Plot

```latex
\begin{tikzpicture}
\begin{axis}[
    xlabel={X Variable},
    ylabel={Y Variable},
    title={Scatter Plot}
]
\addplot[only marks, mark=*] coordinates {
    (1,2) (2,3) (3,1) (4,4) (5,3) (6,5)
};
\end{axis}
\end{tikzpicture}
```

## Multiple Series

```latex
\begin{tikzpicture}
\begin{axis}[
    xlabel={X},
    ylabel={Y},
    legend pos=outer north east
]
\addplot[only marks, mark=*, color=blue] coordinates {
    (1,2) (2,4) (3,3) (4,5)
};
\addlegendentry{Group A}

\addplot[only marks, mark=square, color=red] coordinates {
    (1,3) (2,2) (3,4) (4,3)
};
\addlegendentry{Group B}
\end{axis}
\end{tikzpicture}
```

## Marker Styles

Available marker options:
- `mark=*`: Filled circle
- `mark=o`: Open circle
- `mark=square`: Square
- `mark=triangle`: Triangle
- `mark=x`: Cross
- `mark=+`: Plus sign

## Scatter with Trend Line

```latex
\begin{tikzpicture}
\begin{axis}[
    xlabel={X},
    ylabel={Y}
]
% Data points
\addplot[only marks, mark=*] coordinates {
    (1,2) (2,3.5) (3,4) (4,5.5) (5,6)
};

% Trend line
\addplot[color=red, thick, domain=1:5] {1.1*x + 0.9};
\end{axis}
\end{tikzpicture}
```
