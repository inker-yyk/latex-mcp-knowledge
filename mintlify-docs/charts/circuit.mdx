## Overview

CircuiTikZ is a specialized TikZ library for drawing electronic circuit diagrams. It provides intuitive syntax for components like resistors, capacitors, transistors, and logic gates.

## Basic Circuit

Here's a simple RC circuit example:

```latex
\begin{circuitikz}
\draw (0,0) to[battery, l=$V$] (0,2)
      to[R, l=$R$] (3,2)
      to[C, l=$C$] (3,0)
      to[short] (0,0);
\end{circuitikz}
```

## Common Components

### Resistors and Capacitors

```latex
\begin{circuitikz}
\draw (0,0) to[R, l=$R_1$] (2,0)
      to[C, l=$C_1$] (4,0);
\draw (0,-2) to[vR, l=$R_{var}$] (2,-2)
      to[pC, l=$C_{pol}$] (4,-2);
\end{circuitikz}
```

### Voltage and Current Sources

```latex
\begin{circuitikz}
\draw (0,0) to[battery, v=$12V$] (0,2)
      to[I, i=$I_0$] (3,2);
\end{circuitikz}
```

## Logic Circuits

Create digital logic diagrams:

```latex
\begin{circuitikz}
\node[and port] (and1) at (0,0) {};
\node[or port] (or1) at (2,0) {};
\draw (and1.out) -- (or1.in 1);
\end{circuitikz}
```

## Transistors and Amplifiers

```latex
\begin{circuitikz}
\draw (0,0) node[npn](Q1){};
\draw (Q1.base) to[short, -o] ++(-0.5,0);
\draw (Q1.collector) to[R, l=$R_C$] ++(0,1.5);
\draw (Q1.emitter) to[R, l=$R_E$] ++(0,-1.5);
\end{circuitikz}
```

## Component Labeling

Options for labeling components:

- `l=<text>`: Label above/left
- `l_=<text>`: Label below/right
- `v=<text>`: Voltage label
- `i=<text>`: Current label
- `a=<text>`: Annotation
