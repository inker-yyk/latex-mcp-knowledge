## Overview

These advanced examples demonstrate sophisticated charting techniques, custom styling, and complex visualizations.

## Example 1: Multi-Axis Plot

Create a plot with two y-axes:

```latex
\documentclass{article}
\usepackage{pgfplots}
\pgfplotsset{compat=1.18}

\begin{document}

\begin{tikzpicture}
\begin{axis}[
    xlabel={Time (s)},
    ylabel={Temperature (Â°C)},
    ylabel style={blue},
    ymin=0, ymax=100,
    axis y line*=left
]
\addplot[blue, thick] coordinates {
    (0,20) (1,40) (2,60) (3,80) (4,90)
};
\end{axis}

\begin{axis}[
    xlabel={Time (s)},
    ylabel={Pressure (kPa)},
    ylabel style={red},
    ymin=0, ymax=200,
    axis y line*=right,
    axis x line=none
]
\addplot[red, thick, dashed] coordinates {
    (0,50) (1,100) (2,150) (3,180) (4,190)
};
\end{axis}
\end{tikzpicture}

\end{document}
```

## Example 2: 3D Surface with Color Map

Create a 3D surface plot with custom color mapping:

```latex
\documentclass{article}
\usepackage{pgfplots}
\pgfplotsset{compat=1.18}

\begin{document}

\begin{tikzpicture}
\begin{axis}[
    3d box,
    xlabel={$x$},
    ylabel={$y$},
    zlabel={$z$},
    colormap/hot,
    colorbar,
    view={45}{35}
]
\addplot3[
    surf,
    shader=interp,
    domain=-3:3,
    domain y=-3:3,
    samples=30
] {exp(-x^2-y^2)};
\end{axis}
\end{tikzpicture}

\end{document}
```

## Example 3: Complex Flowchart

Create a decision flowchart with multiple paths:

```latex
\documentclass{article}
\usepackage{tikz}
\usetikzlibrary{shapes.geometric, arrows.meta, positioning}

\begin{document}

\tikzset{
    startstop/.style={rectangle, rounded corners, minimum width=3cm,
        minimum height=1cm, text centered, draw=black, fill=red!30},
    process/.style={rectangle, minimum width=3cm, minimum height=1cm,
        text centered, draw=black, fill=orange!30},
    decision/.style={diamond, minimum width=3cm, minimum height=1cm,
        text centered, draw=black, fill=green!30},
    arrow/.style={thick,->,>=stealth}
}

\begin{tikzpicture}[node distance=2cm]
\node (start) [startstop] {Start};
\node (input) [process, below of=start] {Get Input};
\node (valid) [decision, below of=input, yshift=-1cm] {Valid?};
\node (process) [process, below of=valid, yshift=-1cm] {Process Data};
\node (save) [process, below of=process] {Save Results};
\node (error) [process, right of=valid, xshift=3cm] {Show Error};
\node (retry) [decision, below of=error, yshift=-1cm] {Retry?};
\node (stop) [startstop, below of=save] {End};

\draw [arrow] (start) -- (input);
\draw [arrow] (input) -- (valid);
\draw [arrow] (valid) -- node[anchor=east] {yes} (process);
\draw [arrow] (valid) -- node[anchor=south] {no} (error);
\draw [arrow] (error) -- (retry);
\draw [arrow] (retry) -- node[anchor=south] {yes} ++(0,2) -| (input);
\draw [arrow] (retry) -- node[anchor=east] {no} ++(0,-2) -| (stop);
\draw [arrow] (process) -- (save);
\draw [arrow] (save) -- (stop);
\end{tikzpicture}

\end{document}
```

## Example 4: Parametric 3D Plot

Create a 3D parametric curve (spiral):

```latex
\documentclass{article}
\usepackage{pgfplots}
\pgfplotsset{compat=1.18}

\begin{document}

\begin{tikzpicture}
\begin{axis}[
    3d box,
    view={60}{30},
    xlabel={$x$},
    ylabel={$y$},
    zlabel={$z$}
]
\addplot3[
    blue,
    thick,
    samples=200,
    samples y=0,
    domain=0:720
] ({cos(x)}, {sin(x)}, {x/100});
\end{axis}
\end{tikzpicture}

\end{document}
```

## Example 5: Stacked Area Chart

Create a stacked area chart with filled regions:

```latex
\documentclass{article}
\usepackage{pgfplots}
\pgfplotsset{compat=1.18}
\usepgfplotslibrary{fillbetween}

\begin{document}

\begin{tikzpicture}
\begin{axis}[
    stack plots=y,
    area style,
    xlabel={Year},
    ylabel={Revenue (M\$)},
    legend pos=north west
]
\addplot[fill=blue!30] coordinates {
    (2018,10) (2019,12) (2020,15) (2021,18) (2022,20)
} \closedcycle;
\addlegendentry{Product A}

\addplot[fill=red!30] coordinates {
    (2018,8) (2019,10) (2020,12) (2021,15) (2022,18)
} \closedcycle;
\addlegendentry{Product B}

\addplot[fill=green!30] coordinates {
    (2018,5) (2019,7) (2020,9) (2021,11) (2022,13)
} \closedcycle;
\addlegendentry{Product C}
\end{axis}
\end{tikzpicture}

\end{document}
```

## Example 6: Box Plot

Create a box plot showing statistical distribution:

```latex
\documentclass{article}
\usepackage{pgfplots}
\pgfplotsset{compat=1.18}
\usepgfplotslibrary{statistics}

\begin{document}

\begin{tikzpicture}
\begin{axis}[
    xlabel={Groups},
    ylabel={Values},
    boxplot/draw direction=y
]
\addplot+ [
    boxplot prepared={
        lower whisker=5,
        lower quartile=7,
        median=8.5,
        upper quartile=9.5,
        upper whisker=10
    }
] coordinates {};
\end{axis}
\end{tikzpicture}

\end{document}
```

## Example 7: Network Graph with Custom Styling

Create a styled network diagram:

```latex
\documentclass{article}
\usepackage{tikz}
\usetikzlibrary{positioning, shapes, shadows}

\begin{document}

\tikzset{
    server/.style={cylinder, shape border rotate=90, draw, fill=blue!20,
        minimum height=1.5cm, minimum width=1cm, drop shadow},
    client/.style={circle, draw, fill=green!20, minimum size=1cm, drop shadow},
    connection/.style={thick, ->, >=stealth, shorten >=2pt}
}

\begin{tikzpicture}[node distance=3cm]
\node[server] (server) {Server};
\node[client] (c1) [below left of=server] {C1};
\node[client] (c2) [below of=server] {C2};
\node[client] (c3) [below right of=server] {C3};
\node[client] (c4) [right of=c3] {C4};

\draw[connection] (c1) -- (server);
\draw[connection] (c2) -- (server);
\draw[connection] (c3) -- (server);
\draw[connection] (c4) -- (server);
\draw[connection, red, dashed] (c1) -- node[above] {P2P} (c3);
\end{tikzpicture}

\end{document}
```

## Example 8: Polar Plot

Create a polar coordinate plot:

```latex
\documentclass{article}
\usepackage{pgfplots}
\pgfplotsset{compat=1.18}
\usepgfplotslibrary{polar}

\begin{document}

\begin{tikzpicture}
\begin{polaraxis}[
    title={Polar Plot}
]
\addplot[blue, thick, domain=0:720, samples=100] {1 + sin(x)};
\end{polaraxis}
\end{tikzpicture}

\end{document}
```

## Advanced Techniques

### Custom Color Schemes

```latex
\pgfplotsset{
    custom colormap/.style={
        colormap={custom}{
            rgb255=(0,0,255)
            rgb255=(0,255,0)
            rgb255=(255,255,0)
            rgb255=(255,0,0)
        }
    }
}
```

### Data Processing

```latex
\addplot table[
    x=Time,
    y expr=\thisrow{Value}*1.5+10,
    col sep=comma
] {data.csv};
```

### Animation Frames

```latex
\foreach \angle in {0,10,...,360} {
    \begin{tikzpicture}
    \begin{axis}[view={60}{\angle}]
    \addplot3[surf] {x^2+y^2};
    \end{axis}
    \end{tikzpicture}
}
```

## Performance Optimization

For large datasets:

```latex
\pgfplotsset{
    max space between ticks=40,
    try min ticks=3
}
```
