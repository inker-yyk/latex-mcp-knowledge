## Getting Started

These basic examples demonstrate fundamental charting techniques using TikZ and PGFPlots.

## Example 1: Simple Line Plot

Create a basic line plot showing a linear function:

```latex
\documentclass{article}
\usepackage{pgfplots}
\pgfplotsset{compat=1.18}

\begin{document}

\begin{tikzpicture}
\begin{axis}[
    xlabel={$x$},
    ylabel={$y$},
    title={Linear Function}
]
\addplot[blue, thick] {x};
\end{axis}
\end{tikzpicture}

\end{document}
```

**Output**: A simple line chart showing y = x.

## Example 2: Bar Chart

Create a basic vertical bar chart:

```latex
\documentclass{article}
\usepackage{pgfplots}
\pgfplotsset{compat=1.18}

\begin{document}

\begin{tikzpicture}
\begin{axis}[
    ybar,
    xlabel={Category},
    ylabel={Value},
    symbolic x coords={A,B,C,D},
    xtick=data,
    bar width=20pt
]
\addplot coordinates {
    (A,10) (B,25) (C,15) (D,30)
};
\end{axis}
\end{tikzpicture}

\end{document}
```

## Example 3: Scatter Plot

Create a scatter plot with markers:

```latex
\documentclass{article}
\usepackage{pgfplots}
\pgfplotsset{compat=1.18}

\begin{document}

\begin{tikzpicture}
\begin{axis}[
    xlabel={X},
    ylabel={Y},
    title={Data Points}
]
\addplot[only marks, mark=*, color=red] coordinates {
    (1,2) (2,3) (3,1) (4,4) (5,3) (6,5)
};
\end{axis}
\end{tikzpicture}

\end{document}
```

## Example 4: Simple Flowchart

Create a basic flowchart:

```latex
\documentclass{article}
\usepackage{tikz}
\usetikzlibrary{shapes.geometric, arrows}

\begin{document}

\tikzstyle{box} = [rectangle, draw, fill=blue!20,
    text width=3cm, text centered, minimum height=1cm]
\tikzstyle{arrow} = [thick,->,>=stealth]

\begin{tikzpicture}[node distance=2cm]
\node (start) [box] {Start};
\node (process) [box, below of=start] {Process};
\node (end) [box, below of=process] {End};

\draw [arrow] (start) -- (process);
\draw [arrow] (process) -- (end);
\end{tikzpicture}

\end{document}
```

## Example 5: Two Functions

Plot two functions on the same axis:

```latex
\documentclass{article}
\usepackage{pgfplots}
\pgfplotsset{compat=1.18}

\begin{document}

\begin{tikzpicture}
\begin{axis}[
    xlabel={$x$},
    ylabel={$f(x)$},
    legend pos=north west,
    domain=0:10
]
\addplot[blue, thick] {x};
\addlegendentry{$y=x$}

\addplot[red, thick] {x^2/10};
\addlegendentry{$y=x^2/10$}
\end{axis}
\end{tikzpicture}

\end{document}
```

## Example 6: Simple Node Graph

Create a basic node network:

```latex
\documentclass{article}
\usepackage{tikz}

\begin{document}

\begin{tikzpicture}
\node[circle, draw] (A) at (0,0) {A};
\node[circle, draw] (B) at (2,0) {B};
\node[circle, draw] (C) at (1,2) {C};

\draw (A) -- (B);
\draw (B) -- (C);
\draw (C) -- (A);
\end{tikzpicture}

\end{document}
```

## Required Packages

All examples require these packages:

```latex
\usepackage{tikz}
\usepackage{pgfplots}
\pgfplotsset{compat=1.18}
```

Additional libraries for specific features:

```latex
\usetikzlibrary{shapes.geometric}  % For flowcharts
\usetikzlibrary{arrows}            % For arrow styles
\usetikzlibrary{positioning}       % For node positioning
```

## Tips for Beginners

1. **Start Simple**: Begin with basic examples and add complexity gradually
2. **Use Compatible Version**: Always set `\pgfplotsset{compat=1.18}`
3. **Check Compilation**: Compile frequently to catch errors early
4. **Read Error Messages**: PGFPlots provides helpful error messages
5. **Explore Options**: Try modifying colors, styles, and labels

## Common Mistakes

- Forgetting to load required packages
- Missing semicolons at end of TikZ commands
- Incorrect coordinate syntax
- Not setting compat level for PGFPlots
