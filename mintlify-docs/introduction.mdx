# LaTeX Chart Knowledge Base

A comprehensive, MCP protocol-compliant knowledge base covering 6 major LaTeX packages: TikZ, PGFPlots, CircuiTikZ, Tkz-Euclide, Chemfig, and TikZ-Network.

## What is This?

This knowledge base contains **6,812 structured knowledge items** extracted from 6 official LaTeX manuals, including:

- **5,325 executable examples** - Ready-to-use LaTeX code for charts, circuits, geometry, chemistry, and networks
- **1,103 command specifications** - Detailed syntax and parameter documentation
- **315 component definitions** - Circuit elements, geometric shapes, and more
- **69 other items** - Environments, key-value pairs, and best practices
