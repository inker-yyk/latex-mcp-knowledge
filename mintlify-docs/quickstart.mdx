## Option 2: Use the MCP API

For programmatic access or AI model integration:

### Step 1: Get an API Key (Optional)

```bash
# Currently, the API is open access
# Future versions may require authentication
```

### Step 2: Make Your First Request

<CodeGroup>
```bash cURL
curl "https://your-domain.mintlify.app/mcp/api/mcp/latex/chart/example?chart_type=line_chart&limit=5"
```

```python Python
import requests

response = requests.get(
    "https://your-domain.mintlify.app/mcp/api/mcp/latex/chart/example",
    params={
        "chart_type": "line_chart",
        "limit": 5
    }
)

examples = response.json()
print(examples["data"][0]["content"]["code"])
```

```javascript JavaScript
const response = await fetch(
  'https://your-domain.mintlify.app/mcp/api/mcp/latex/chart/example?chart_type=line_chart&limit=5'
);

const examples = await response.json();
console.log(examples.data[0].content.code);
```
</CodeGroup>

## Option 3: Integrate with Claude/GPT

### For Claude Desktop

Add this MCP server to your Claude Desktop configuration:

```json
{
  "mcpServers": {
    "latex-charts": {
      "url": "https://your-domain.mintlify.app/mcp",
      "description": "LaTeX chart generation knowledge base"
    }
  }
}
```

### For Custom Models

Use our API as a retrieval source for your LLM:

```python
def get_latex_examples(chart_type: str, limit: int = 5):
    """Retrieve LaTeX examples for a given chart type"""
    response = requests.get(
        "https://your-domain.mintlify.app/mcp/api/mcp/latex/chart/example",
        params={"chart_type": chart_type, "limit": limit}
    )
    return response.json()["data"]

# Use in your LLM prompt
examples = get_latex_examples("line_chart")
prompt = f"Generate a line chart. Here are official examples:\n{examples}"
```

## Common Use Cases

### 1. Find a Specific Command

```bash
curl "https://your-domain.mintlify.app/mcp/api/mcp/latex/chart/command?command_name=addplot"
```

### 2. Search All Knowledge

```bash
curl "https://your-domain.mintlify.app/mcp/api/mcp/latex/chart/search?q=axis+labels"
```

### 3. Get Best Practices

```bash
curl "https://your-domain.mintlify.app/mcp/api/mcp/latex/chart/feedback?feedback_type=warning"
```

```json

  "data": [

      "id": "abc123",
      "type": "executable_example",
      "macro_package": "pgfplots",
      "metadata": {
        "chart_type": "line_chart",
        "tags": ["pgfplots", "line_chart", "example"]

      "content": {

        "parameters": []
      }
    }
  ],
  "meta": {
    "total": 1,
    "count": 1,
    "query_time_ms": 12.34
  }
}
```
