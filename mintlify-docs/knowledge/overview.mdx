## About This Knowledge Base

This knowledge base provides comprehensive information about creating charts, diagrams, and visualizations in LaTeX using TikZ and PGFPlots.

## What's Included

### TikZ Basics
- Node and edge creation
- Positioning and layout
- Styling and customization
- Path operations
- Coordinate systems

### PGFPlots
- 2D plotting (line, bar, scatter)
- 3D plotting and surfaces
- Axis customization
- Data visualization
- Color maps and legends

- [Line Charts](../charts/line-chart)
