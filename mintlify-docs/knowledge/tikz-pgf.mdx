# TikZ/PGF Package

**TikZ/PGF** is the foundational graphics package for LaTeX, providing comprehensive tools for creating high-quality vector graphics programmatically.

## Package Information

- **Package Name**: tikz-pgf
- **Items in Knowledge Base**: 3,696
- **Manual Version**: v3.1.10
- **Primary Use**: General-purpose graphics, diagrams, and visualizations

## What is TikZ/PGF?

**PGF** (Portable Graphics Format) is the lower-level system that provides basic drawing commands. **TikZ** (TikZ ist kein Zeichenprogramm) is a higher-level, user-friendly interface built on top of PGF.

### Key Features

- **Nodes and Paths**: Create complex node-based diagrams
- **Coordinate Systems**: Cartesian, polar, relative positioning
- **Shapes Library**: Circles, rectangles, polygons, custom shapes
- **Styling System**: Powerful style definitions and inheritance
- **Transformations**: Rotate, scale, shift, clip
- **Graph Drawing**: Automatic graph layout algorithms
- **3D Graphics**: Basic 3D coordinate system support

## Common Use Cases

### 1. Node Diagrams

```latex
\begin{tikzpicture}
  \node[circle, draw] (A) at (0,0) {A};
  \node[circle, draw] (B) at (2,0) {B};
  \draw[->] (A) -- (B);
\end{tikzpicture}
```

### 2. Flowcharts

```latex
\begin{tikzpicture}[node distance=2cm]
  \node[rectangle, draw] (start) {Start};
  \node[diamond, draw, below of=start] (decision) {Decision};
  \node[rectangle, draw, below left of=decision] (left) {Option A};
  \node[rectangle, draw, below right of=decision] (right) {Option B};
\end{tikzpicture}
```

### 3. Mathematical Diagrams

```latex
\begin{tikzpicture}
  \draw[->] (-0.5,0) -- (4,0) node[right] {$x$};
  \draw[->] (0,-0.5) -- (0,4) node[above] {$y$};
  \draw[domain=0:3.5, smooth, variable=\x, blue] plot ({\x}, {\x*\x/4});
\end{tikzpicture}
```
