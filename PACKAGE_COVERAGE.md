**解决方案**：
- 为 `.dtx` 格式手册开发专门提取器
- 为 tikz-qtree 添加对 `SideBySideExample` 和 `Verbatim` (大写) 环境的支持
- 考虑从 PDF 或 README 中手动提取关键信息
